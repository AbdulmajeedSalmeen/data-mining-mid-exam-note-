Mid Exam Study Notes

Module 1: Data Mining Overview

What Is Data Mining?

Data mining is a process of extracting knowledge from data regardless of the reasons and what this knowledge is used for.

Knowledge Discovery (KDD) Process

  • Data cleaning (to remove noise and inconsistent data)
  • Data integration (where multiple data sources may be combined)
  • Data selection (where data relevant to the analysis task are retrieved)
  • Data transformation (summary or aggregation operations)
  • Data mining (where intelligent methods are applied to extract patterns)
  • Pattern evaluation (to identify interesting patterns)
  • Knowledge presentation (visualization techniques used to present knowledge)

Summary Table for Module 1

Key Concept Description
Data Cleaning Removes noise and inconsistent data.
Data Mining Application of intelligent methods to extract data patterns.
Pattern Evaluation Identify interesting patterns in the data.
Data Types Mined Relational database, data warehouse, transactional database.

Module 2: Data Quality

Why Preprocess the Data?

Preprocessing improves data quality including accuracy, completeness, consistency, and interpretability.

Major Tasks in Data Preprocessing

  • Data cleaning
  • Data integration
  • Data reduction
  • Data transformation

Summary Table for Module 2

Preprocessing Task Description
Data Cleaning Removing noise, handling missing and inconsistent data.
Data Integration Combining data from multiple sources into one coherent data store.
Data Reduction Reducing the size of the data without losing valuable information.
Data Transformation Converting data into a format suitable for analysis.

Module 3: Data Reduction Strategies

What Is Data Reduction?

Data reduction is the process of obtaining a smaller representation of a dataset that retains the same or nearly the same analytical results.

Data Reduction Strategies

  • Dimensionality reduction
  • Wavelet transforms
  • Principal Components Analysis (PCA)
  • Feature subset selection
  • Numerosity reduction
  • Data compression

Summary Table for Module 3

Reduction Strategy Description
Dimensionality Reduction Reducing the number of attributes or features in the dataset.
Wavelet Transform Decomposing data into different frequency sub-bands.
PCA Finds projections that capture the largest variance in the data.
Feature Subset Selection Selecting a subset of relevant features for analysis.
